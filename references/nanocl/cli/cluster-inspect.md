<h2>Name</h2>

nanocl-cluster-inspect - Inspect cluster by its name

<h2>Synopsis</h2>

**nanocl-cluster-inspect** \[**-h**\|**\--help**\] \<*NAME*\>

<h2>Description</h2>

Get cluster information from its name in current namespace

<h2>Options</h2>

**-h**, **\--help**

:   Print help information

\<*NAME*\>

:   name of the cluster to inspect
